# Portfolio - S.Païder Oba Ibara

## Description

Portfolio personnel de S.Païder Oba Ibara, Ingénieur en Génie Numérique. Le site présente mon parcours, mes compétences et mes projets réalisés.

## Fonctionnalités

- Présentation avec photo et titre
    
- Section "À propos" avec mon parcours
    
- Compétences techniques avec barres de progression
    
- Galerie de 7 projets
    
- Timeline d'expérience professionnelle
    
- Formulaire de contact
    
- Responsive sur tous les écrans
    
- Design sombre
    

## Technologies

- HTML5
    
- CSS3 (variables, animations, transitions)
    
- JavaScript (menu burger, scroll)
    
- Font Awesome (icônes)
    
- Google Fonts (police Inter)

portfolio/

## Installation

1. Clone le repo
    
2. Ouvre `index.html` dans ton navigateur
    
3. Pas besoin d'installation supplémentaire
    

## Personnalisation

Les couleurs principales sont dans `style.css` :

```
:root {
  --color-primary: #FF6B35;
  --color-secondary: #00D4FF;
}
```

Pour modifier les infos perso, édite `index.html` :

- Nom et titre dans la section Hero
    
- Description dans About
    
- Compétences dans Skills
    
- Projets dans Projects
    
- Expériences dans Experience
    
- Coordonnées dans Contact
    

## Déploiement

### GitHub Pages

1. Settings > Pages
    
2. Sélectionne la branche main
    
3. Le site est en ligne
    

### Netlify / Vercel

1. Connecte ton dépôt GitHub
    
2. Déploiement automatique
